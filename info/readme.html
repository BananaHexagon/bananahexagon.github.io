召喚方法

必要なもの

品目 分量 備考
道徳の教科書 1500ページ程度 日本のものが望ましい
レインコート 3着
ほうき 4本 新品
木炭 12st 過去に個人間を譲渡されたことのないもの
碧砂厳市の土 500g以上 多ければ多いほどよい
熱湯 40L以上 土を増やす場合はこちらも増やす
リムジン 1台 車種は問わない

(1)魔法陣

熱湯に土を溶かし、魔法陣を描きます。

(2)器作り

教科書を頭、ほうきを手足、木炭を胴体に見立てて器を作り、レインコートを着せます。

(3)配置

(1)の魔法陣の中央に(2)で作った器を置き、80m程離れたところにリムジンを配備します。

(4)仕上げ

リムジンで器を轢きます。

結果

色違いのボルケニオンゲット！
